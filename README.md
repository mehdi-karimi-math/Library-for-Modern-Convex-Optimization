# Library-for-Modern-Convex-Optimization
This library provides benchmark instances for classes of convex optimization problems that have recently gained practical attention. Currently, it includes two major categories: 
- Hyperbolic programming: 
  The banchmark library for hyperbolic programming in based on [this paper](https://arxiv.org/abs/2602.23260). Please go the above folder `Hyperbolic Programming` for details and creating the results in the paper. 
- Quantum relative entropy (QRE) programming:
  The benchmark for QRE is based on [this paper](https://pubsonline.informs.org/doi/abs/10.1287/ijoc.2024.0570).
